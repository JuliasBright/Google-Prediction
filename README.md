### Installing

Quandl

A python module which provides various  Data for different functionalities. 

On Windows-

  pip install quandl
  
On Mac-
  
  sudo pip install quandl

Numpy. 

On Windows-

  pip install numpy
  
On Mac-
  
  sudo pip install numpy

Sklearn. 

On Windows-

  pip install sklearn
  
On Mac-
  
  sudo pip install sklearn
